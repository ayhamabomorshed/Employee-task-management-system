# 🚀 Task Management System – Java Swing GUI Prototype

"From idea… to interactive screens 👀"

This project is a **Task Management System prototype** built using **Java Swing (NetBeans)**.  
It represents the GUI phase of the system development process, where wireframes were transformed into working graphical interfaces.

---

## 📌 Project Overview

The system simulates a basic task management platform that includes different roles such as:
- Admin
- Employee

It focuses on building a complete **GUI prototype** before integrating backend logic and database functionality.

---

## 💻 Features Implemented

### 🔐 Login System
- User authentication (Admin / Employee)
- Input validation
- Terms & conditions checkbox control

### 🧑‍💼 Admin Dashboard
- Create employee accounts
- Add and manage tasks
- View task management table
- Navigation between system modules

### 👨‍💻 Employee Module
- View assigned tasks
- Update task status (Done / Pending / In Progress)
- Interactive task selection

### 📊 Reports Dashboard
- Display total tasks
- Show completed and in-progress tasks
- Employee performance overview table

### 👥 Employee Information Page
- Display employee details in a table
- Navigation to Admin and Reports pages

---

## 🧠 Key Learning Outcomes

- Converting wireframes into Java Swing GUI applications
- Managing multiple JFrame windows in a single project
- Designing user-friendly interfaces
- Handling basic input validation
- Improving UI navigation flow between screens
